Welcome to CI demo!

If you see this page, all servers to compose a CI system
is successfully installed and working.

This CI system consists of the following servers

  • Gerrit works as a web based code review and Git version control system.
  • Jenkins CI works as an automatic building and testing server which is the core component of the CI system.
  • Redmine works as a web based issues and tasks tracking system.
  • Sonatype Nexus works as a local Maven repository.

Gerrit and Redmine share the same LDAP authentication system which is specified in the config file. So you are able to use the same username and password to log in those systems.

A DEMO project is pre-imported into Gerrit , Jenkins and Redmine to demonstrate how to connect these systems together. Feel free to play around with this project. The whole system can be destroyed or recreated by the destroyContainer.sh or the recreate.sh.

If you are interested in administrating these servers, the default credentials of super user are listed as below.
Gerrit: admin/passwd
Redmine: admin/admin
Nexus: admin/admin123
